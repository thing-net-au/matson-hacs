REPOSITORY READY TO PUSH!

All code is committed and ready. Just need to:

1. Create repo at: https://github.com/new
   Name: matson-assistant
   Public: YES
   Don't initialize with anything

2. Then run: git push -u origin main

3. Create tag: git tag -a v1.0.0 -m "Initial release"
   Push tag: git push origin v1.0.0

4. Create release on GitHub from the tag

5. Add to HACS in HomeAssistant

See PUSH_TO_GITHUB.md for detailed instructions.
